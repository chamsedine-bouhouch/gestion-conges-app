## Installation

1. install laravel dependencies
   
`composer install`

2. copy environment file
   
`cp .env.example .env`

3. generate application key
   
`php artisan key:generate`

4. install node packeges
   
`npm install`

5. run the project
   
- `php artisan serve`
- `npm run dev`
